# 🗓️ Login App / Reserva Web

Aplicación web creada con **Next.js** que permite a los usuarios realizar reservas y guardar los datos directamente en una hoja de **Google Sheets**, utilizando una API segura.

---

## 🚀 Tecnologías utilizadas

- **Next.js 14**
- **React**
- **TypeScript**
- **Google Sheets API**
- **Vercel** → para el despliegue
- **GitHub** → control de versiones
- **Supabase** → para autenticación

---

## ✨ Funcionalidades

✅ Formulario de reserva con los campos:
- Nombre  
- Email  
- Cantidad de personas  
- Fecha  
- Hora  

✅ Envío automático de los datos al documento de Google Sheets  
✅ Validación básica de los campos del formulario  
✅ Redirección al inicio tras enviar la reserva correctamente
